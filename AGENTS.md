# AGENTS.md

## Setup
- Create Python environment: `conda env create -f environment.yml`
- Activate environment: `conda activate leetcode`
- Update environment: `conda env update -f environment.yml --prune`

## Running Solutions
- Python solutions: Run with `python filename.py`
- TypeScript solutions: Run with `bun filename.ts`

## Project Structure
- Problems organized by difficulty: `easy/`, `medium/`, `hard/`
- Data science problems in `datascience/` folder
- Each problem has a template: `template.py`

## Important Notes
- Solutions use Python by default, with select problems in TypeScript, JavaScript, and C#
- Problems are tracked in the main README.md with date, difficulty, and source
- Focus is currently on 'Easy' problems
- All problems have accompanying test cases in their respective files
